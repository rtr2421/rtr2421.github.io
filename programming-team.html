Programming Team

Installing Software:

  1. Download WPILib & VSCode.
  2. Create an account on Github, and give your username to a programming mentor to be added to our group.
  3. Install Github Desktop.
  4. Install Pathplanner to create autonomous paths.

If you are interested in learning how to use Github, please see this Github tutorial.

Important links:

  • Getting started with FRC development.
